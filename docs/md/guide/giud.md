# 介绍

---

`lin-wx-ui` 是一款基于 `原生微信小程序` 的前端 UI 组件库，主要集成了我平时在开发中使用到的 UI 组件

## 预览

## 特性

- 基于 `原生微信小程序` 开发的 UI 组件
- 使用 npm + gulp 的工作流
- 提供友好的 API，可灵活的使用组件
- 支持按需引入，减少项目打包体积
- 偏向于业务组件
- 提供完善的文档
