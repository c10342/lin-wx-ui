# 介绍

---

`lin-wx-ui` 是一款基于 `原生微信小程序` 的前端 UI 组件库，主要集成了我平时在开发中使用到的 UI 组件

## 预览

扫描下方小程序二维码，体验组件库示例：

![qrcode](/images/qrcode.jpg)

## 特性

- 基于 `原生微信小程序` 开发的 UI 组件库
- 使用 `typescript` 编写
- 使用 npm + gulp 的工作流
- 提供单元测试用例
- 提供完善的使用文档
